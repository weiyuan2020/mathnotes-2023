# Stokes' theorem


### thm:10.33 
 If $\Psi$ is a $k$-chain of class
$\mathscr{C}''$ in an open set $V \subset \mathbb{R}^m$ and if $\omega$ is a
$(k-1)$-form of class $\mathscr{C}'$ in $V$, then 
\label{eq:10.91}

$$

        \int_{\Psi} \mathrm{d} \omega = 
        \int_{\partial \Psi} \omega .
$$
